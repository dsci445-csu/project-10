# Reproduce

We copy and pasted that code into slides.Rmd and paper.Rmd as they were needed. All necessary code now resides in both slides.Rmd and paper.Rmd. You can look at the other files if you're curious, but some of them are kind of a mess as doing this project in multiple files kinda defeats the whole purpose of git, but it's whatever.

To reproduce our presentation slides, knit slides.Rmd to a PDF.

To reproduce our paper, knit paper.Rmd to a PDF.
